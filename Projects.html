CodeCade

CodeCade is an interactive web platform that introduces children to coding through gameplay and real-time code visualization. Kids play simple games like Rock Paper Scissors or Hangman on one side of the screen, while the code behind the game is shown and highlighted live on the other side, showing exactly what part of the code is executing during each action. The site also includes an AI assistant that explains each line of code in kid-friendly language.

Tech Stack
  • Front-end: Next.js, Tailwind CSS
  • Back-end: Django
  • AI Integration: OpenAI API
  • Design: Figma
  • Project Management: GitHub Projects (templates, issue tracking, branches)
